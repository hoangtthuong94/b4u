Training targets

- The goal of the program is to help students become acquainted with the knowledge and skills required to be performance testing specialists.
- Approach tools are being used in the market for performance testing.

Knowledge and skills gained after the course

- Understand the overview of performance testing
- Overview of various types of performance testing tools such as: Performance Test, Stress Test, API testing.
- JMeter configuration in Jenkins on the Windows / Linux server.
- Use the JMeter tool for performance testing web app and mobile app.
- Analyze and synthesize results to create complete report.
- Join the project with trainer.

Career opportunities

- Become a Performance Tester

Training period

- Training time is 33 hours.
- The schedule is Monday - Wednesday - Friday, from 18:00 to 20:30.
- Fee: 3.900.000VNĐ
- For Students: - 10%

Target students

- Those who look forward to working in the software testing as performance tester.

Input requirements

- Familiar with the working environment on the Windows operating system, Microsoft Office.
- Basic knowledge of programming languages Java, JavaScript, Beanshell
- Familiar with the Linux operating system environment (such as Fedora, CentOS, Ubuntu...) at the level of understanding and using the command line on Linux.
- Can read and understand English.
